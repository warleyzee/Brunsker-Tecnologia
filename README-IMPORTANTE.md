# Brunsker-Tecnologia
Trabalho elaborado para o teste de front-end na Brunsker Tecnologia.

Olá, aqui contém algumas informações necessárias para execução do projeto:

Fiz o teste que me foi enviado e quis demonstrar um pouco do meu conhecimento. 

Criei o projeto em PHP e usando o banco de dados para armazenar as informção de cadastro e login. 

Para utilizar o BD segue o link para fazer o donwload do XAMPP:

https://www.apachefriends.org/xampp-files/7.2.26/xampp-windows-x64-7.2.26-0-VC15-installer.exe

Com o XAMPP instalado coloque todo o projeto no diretorio --> C:\Program Files (x86)\xampp\htdocs

Dê start em "Apache" e "Mysql" no XAMPP (dois cliques no icone perto do relogio).

No navegador do browser digite " http://localhost/phpmyadmin "

Na aba "SQL" criei o banco de dados  --> CREATE DATABASE brunsker --> executar 

Selecione o BD criado e clique na aba importar --> Escolher arquivo -->  C:\Program Files (x86)\xampp\htdocs\Brunsker_Tecnologia\bd\brunker.sql --> executar 

Feito isto, o projeto esta apto para ser usado.
